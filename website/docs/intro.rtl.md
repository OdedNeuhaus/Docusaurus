# Simple Documentation
במשך זמן רב היה לנו בצוות Watson את החזון להרים תיעוד מקיף ומעמיק לSimple.
השימוש בSimple רק הולך וגדל עם הזמן, והחלטנו שזה הזמן הנכון להגשים את החזון שלנו.
**אז ברוכים הבאים לתיעוד הרשמי של Simple!**
## על התיעוד עצמו
התיעוד מתאים למשתמשי Simple בכל הרמות, התיעוד ידבר על האובייקטים הבסיסיים ביותר של Simple, וגם על היותר מורכבים.
נלמד יחד על האובייקטים השונים במערכת, ועל אופציות הקינפוג השונות שלהם, נלמד על איך ניתן לנהל את תכולות הSimple שלכם, וניגע בכל נושא שיכול להיות רלוונטי עבורכם - משתמשי Simple.
אך לפני הכל, וודאו שיש לכם Simple מותקנת על העמדה:
```bash
pip install simple
```
