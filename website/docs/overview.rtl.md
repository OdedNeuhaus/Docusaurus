# קצת על Simple
## מה זה Simple?

Simple היא CLI דרכו נוכל לפרוס את הקוד שלנו בסנטרדט מבצעי בקלי קלות.

מה זה אומר "סטנדרנט מבצעי"?:
- פרוס על Kubernetes באופן בו אתם בוחרים.
- כל ריצת Simple מקבלת במתנה dashboard בGrafana שמכיל מידע ניטורי רב על הריצות שלכם.
- כל הכיף הזה מנוהל במקום אחד - הCLI של Simple.
